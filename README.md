# Buoyance - Nigerian Tax Compliance Platform

A web platform for individuals, businesses, and accountants to manage Nigerian tax compliance (CIT, PIT, VAT, WHT, TCC).

## Getting started

Copy `.env.example` to `.env` and fill in your values before running the app.
