# Brand Guidelines: Buoyance

## Visual Identity
- **Primary Color**: Violet / Deep Purple
- **Accent Color**: Neon / Vibrant Green (for highlights and actions)
- **Typography**: Modern, professional sans-serif (shadcn defaults)

## Principles
- **Clarity**: Financial data must be easy to read and unambiguous.
- **Trust**: Consistent UI/UX to ensure users feel their sensitive tax data is handled securely.
- **Modernity**: Leverage glassmorphism, smooth transitions, and a premium 2026-standard aesthetic.

## Usage
- Prefer semantic theme classes (`bg-primary`, `text-accent`) over hardcoded hex values.
- Ensure high contrast for accessibility on financial tables and forms.
